# 🐞 BUG01 - Sistema não apresenta mensagem de erro com credenciais inválidas

## 📌 Descrição 
O sistema não informa ao usuário quando são utilizadas credenciais inválidas durante o login.

## ✅ Comportamento esperado
O sistema deve apresentar uma mensagem de erro informando que as credenciais são inválidas.

## ❌ Comportamento atual
O sistema não apresenta nenhuma mensagem de erro ao usuário.

## 🔁 Passos para reproduzir
Acessar página de login

Informar credenciais de login inválidas

Clicar no botão de login

## ⚠️ Severidade
Alta 

## 🎯 Prioridade
Alta

US02 - Login
